# 🎯 InterviewAce

InterviewAce is a simple Command Line Interface (CLI) quiz application built using Python. It helps users practice basic interview questions on Python, Git, and Linux while keeping track of their score and performance.

---

## 🚀 Features

- 📚 Three interview categories:
  - Python
  - Git
  - Linux

- 🎲 Random question selection

- ✅ Checks whether the answer is correct

- ❌ Displays the correct answer if the user is wrong

- 📊 Tracks:
  - Correct Answers
  - Questions Attempted
  - Final Percentage

- 🔁 Menu-driven interface

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Radhikashekar/InterviewAce.git
```

2. Open the project folder

```bash
cd InterviewAce
```

3. Run the application

```bash
python app.py
```

or

```bash
py app.py
```
